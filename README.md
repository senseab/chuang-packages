## 食用方法 🍜

### 前置条件

1. 安装 [Nix](https://nixos.org/download.html)
2. 准备一颗强大的心脏 ❤️🔧

### 快速开始

```bash
# 添加本仓库通道
nix-channel --add https://github.com/your-repo/your-channel

# 进入量子叠加态环境
nix-shell -p \
    uos-print-daemon \      # 永远在重启的打印服务
    kylin-secure-module \   # 薛定谔的安全组件
    neo-kylin-db-client \   # 需要先证明自己是root才能用的数据库客户端
    --show-trace
```
